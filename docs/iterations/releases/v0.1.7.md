# AtomBox v0.1.7 迭代计划

> 文档状态：Draft
>
> 创建时间：2026-07-23
>
> 范围：增强远程文件搜索与下载交互，并保留暂不开放的批量文件操作实现。

## 版本目标

v0.1.7 改善远程文件浏览器的日常操作：搜索框支持按 Enter 执行搜索，清空搜索后自动恢复完整列表；单文件下载不再跳转传输队列。文件多选、批量下载和批量删除已经实现，但本版本暂不开放对应 UI。

## 非目标

- 不支持批量选择、下载文件夹或 Bucket。
- 不新增 Provider 批量删除协议。
- 不修改 Provider、Transfer 或 Infrastructure 的公共契约。
- 不跨目录、翻页或刷新保留选择状态。
- 本版本不向用户展示文件选择列或批量操作栏。
- 本版本不增加 FloatButton Tooltip。

## 验收标准

- 搜索框输入关键字后按 Enter 与点击搜索按钮行为一致。
- 点击清除按钮或手工清空搜索框后，当前目录自动恢复完整列表。
- 单文件下载任务创建后保持当前页面，通过顶部 Message 和传输队列数字反馈状态。
- 传输队列数字只从活动任务快照统计，不因任务创建通知重复累加。
- OSS、FTP/SFTP 和 WebDAV 文件列表均不显示选择列、复选框或批量操作栏。
- 隐藏多选 UI 后文件列表不残留空白列或多余间距。
- 左侧导航顶部背景与菜单背景一致，缩略图列与列表边框保留合理间距。
- 完整构建和自动化测试通过，Desktop 完成手工 smoke test。
