# AtomBox Desktop 左侧菜单

> 文档状态：第一版发布基线冻结
>
> 冻结时间：2026-06-22
>
> 冻结范围：当前文档定义的 Desktop 左侧菜单结构、菜单层级、动态账号节点和点击行为
>
> 变更规则：第一版发布前代码已冻结；如需调整本文档定义的菜单结构或交互边界，必须先说明原因，再同步更新相关 UI 和 Presentation 文档。

本文档记录 AtomBox Desktop 第一版左侧 Navigation Tree 的菜单结构和点击行为。本文档只定义低保真信息架构，不定义具体颜色、字体、间距或图标尺寸。

## 1. 菜单原则

左侧菜单服务于 Desktop 主工作区导航。

- 一级菜单表达高层资源域。
- 远程存储下包含总入口、资源类型分组和真实账号节点。
- 远程存储下不再提供独立的“添加远端存储”动作节点；新增账号入口由远程存储空状态、汇总页右下角 FloatButton 或账号管理页提供。
- 资源类型分组节点 `OSS`、`FTP/SFTP`、`WebDAV` 可以点击进入对应分类入口，也可以展开查看该分类下的真实账号。
- 左侧菜单不承载远程目录树。
- 左侧菜单不承载复杂筛选条件。
- 左侧菜单不承载新增、编辑账号表单。

第一版删除“工具”一级菜单，暂不提供诊断页面。日志查看、环境信息、导出诊断信息等诊断能力不进入第一版左侧菜单。

## 3. 节点类型

左侧菜单包含五类节点：

| 节点类型 | 示例 | 职责 |
| --- | --- | --- |
| 首页节点 | 首页 | 打开 Desktop 首页。 |
| 资源域节点 | 远程存储、传输、设置 | 表达高层工作域；远程存储节点同时可进入远程存储汇总首页。 |
| 资源类型分组节点 | OSS、FTP/SFTP、WebDAV | 展开或折叠该类型账号列表；点击时进入对应分类入口页。 |
| 功能入口节点 | 传输队列、历史记录、应用设置、账号管理 | 点击后切换右侧 Content Area。 |
| 连接实例节点 | Aliyun OSS - production、SFTP - dev-server | 点击后打开远程浏览页，并加载该连接的默认路径。 |

`OSS`、`FTP/SFTP`、`WebDAV` 是远程存储下的资源类型分组节点，不是具体 provider 实例。

具体 provider 来自账号配置中的 provider 类型和显示名称，不在左侧菜单中写死。

## 4. 点击行为

| 节点 | 行为 |
| --- | --- |
| 首页 | 右侧打开首页。 |
| 远程存储 | 右侧打开远程存储汇总首页；该页展示全部已添加账号，按对象存储、FTP/SFTP 和 WebDAV 分组。 |
| OSS | 右侧打开 OSS 分类入口页；展示 OSS 账号列表或无账号空状态。 |
| FTP/SFTP | 右侧打开 FTP/SFTP 分类入口页；展示 SFTP/FTP 账号列表或无账号空状态。 |
| WebDAV | 右侧打开 WebDAV 分类入口页；展示 WebDAV 账号列表或无账号空状态。 |
| 具体连接实例 | 右侧打开远程浏览页，并加载该连接的默认路径或上次路径。 |
| 传输 | 展开或折叠传输分组。 |
| 传输队列 | 右侧打开传输队列页面。 |
| 历史记录 | 右侧打开传输历史页面。 |
| 设置 | 展开或折叠设置分组。 |
| 应用设置 | 右侧打开应用设置页面。 |
| 账号管理 | 右侧打开账号管理页面。 |

## 5. 页面边界

第一版不在左侧菜单提供传输状态筛选。

如果后续增加以下筛选，只能放在右侧页面内部，不能进入左侧菜单：

- 全部。
- 上传中。
- 下载中。
- 失败。
- 已完成。
- 已取消。

诊断相关能力不进入第一版左侧菜单。

左侧菜单负责“去哪里”，右侧页面负责“怎么看、怎么筛选、怎么操作”。

## 6. 设计约束

- 第一版不做左侧远程目录树。
- 远程存储分类下可以显示连接实例节点，但不能显示远程目录和文件。
- 具体连接实例可以直接触发远程浏览加载。
- 新增和编辑账号必须通过统一账号弹窗完成。
- 左侧菜单不直接调用 Provider、Transfer Runtime 或 Infrastructure。
- 左侧菜单点击行为必须通过 Desktop 内部 NavigationService 切换右侧 Content Area。

## 7. AtomUI 使用

| 操作 | 规则 |
| --- | --- |
| 竖向菜单 | 使用 AtomUI 的 NavMenu / Menu 系列组件承载。 |
| 菜单图标 | 使用 AtomUI 内置 Ant Design 图标库。 |
| 长名称处理 | 左侧菜单宽度可拖动，provider 名称过长时使用菜单宽度和文本裁剪处理。 |
